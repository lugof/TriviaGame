# TriviaGame


### About the game:

```
Welcome to my trivia game. 
1.- Select Start to start playing the game.
2.- This game has 7 trivia questions. You will see 4 possible answers.
3.- Possible answers will behave as a button so you can click on any of them to select them.
4.- As soon as you click on 1 of them the screen will tell you if you are right or wrong.
5.- If you don't click on any answer, the timer will go down to 0 and tell you that your time is up, showing you the right answer.
6.- Once you have completed all 7 questions, a gameover screen will appear, letting you know how many were right, wrong and unanswered.
7.- You will also see a button to start the game over if you choose to.
```

### Enjoy the game!
